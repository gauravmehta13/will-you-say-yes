# Product Overview

An interactive web-based joke application that playfully asks "Will you go out with me?" The app features a humorous interaction where the "No" button moves away when users try to click it, while the "Yes" button grows larger with each attempt. The experience is designed to be lighthearted and entertaining, ultimately encouraging users to click "Yes" through playful UI manipulation.

## Core Features
- Interactive button behavior with dynamic positioning
- Progressive UI changes (button scaling and text updates)
- Mobile and desktop support
- Celebratory success state with animated GIF
